# seteuk-harness

수행평가 학생 원본으로 과목별 세부능력 및 특기사항(과세특) 초안을 생성하는
Claude Code 스킬입니다. 쓰면 쓸수록 사용자의 문체와 활동에 맞춰 정교해집니다.

## 특징
- 규칙 인터뷰 + 구조 샘플 시연: 학생 원본을 분석해 세특 구조 후보를 실제 완성 문장으로 비교 선택
- 2단 검증 게이트: 바이트·금지어·문두는 스크립트가, 근거 충실성·개별성·톤 일치는 LLM 루브릭이 검사.
  검증을 통과해야만 xlsx가 저장됩니다
- 성장하는 위키: 회차마다 인터뷰 결정·채택 구조·검수 수정 패턴이 wiki에 쌓여
  다음 회차의 인터뷰가 짧아지고 표현이 사용자 문체에 수렴합니다

## 설치
1. 이 저장소를 `C:\Users\<사용자>\.claude\skills\seteuk-harness`(또는 `~/.claude/skills/seteuk-harness`)에 클론합니다
2. Python 3.10+ 와 openpyxl이 필요합니다: `python -m pip install openpyxl`
3. Claude Code에서 "세특 작성해줘" 또는 "과세특 생성"이라고 요청하면 스킬이 동작합니다

## 사용 흐름
학생 원본 위치 제시 → 규칙 인터뷰(5문항 이내) → 구조 시연에서 선택 →
생성·검증 → xlsx 초안 수령 → 검수 후 수정본을 주면 회고가 위키에 학습

## 데이터 경계 (중요)
- 스킬 위키에는 학생 실명·학번·원문이 기록되지 않습니다(익명 통계만)
- 학생 데이터가 담긴 산출물(xlsx, 활동프로파일.json)은 전부 여러분의 작업 폴더에 남습니다
- `wiki/활동이력/`은 .gitignore로 커밋에서 제외됩니다
- `wiki/문체관찰.md`와 구조패턴·표현사전·루브릭의 개인 사용 중 변경분은 추적되는 파일이므로, 포크·공유 저장소에 푸시하기 전에 커밋하지 말고 로컬에만 두세요
