# 구조패턴

구조 심층 인터뷰에서 시연할 패턴 목록. 회고 단계가 패턴별 사용 실적을 갱신한다.
시연 규칙: 표본 학생 1명의 원문으로 후보 패턴별 완성 세특을 만들어 나란히 보여준다.

## P-01 강점중심 3단
- 구성: 활동 개요(소재 선정 관점) 1문장 → 학생별 개별 강점 2~3문장(원문 근거 인용) → 성장·역량 마무리 1문장
- 어울리는 활동: 산출물에 학생별 편차가 크고 개별성 부각이 중요한 활동(비평문, 감상문, 에세이)
- 유의점: 강점은 학생마다 다른 지점을 골라야 같은 소재라도 세특이 구별된다
- 사용 실적: 2026-07 소설 비평하기(2개 학급)에서 채택

## P-02 과정서술형
- 구성: 활동 개요 → 수행 과정 추적(선정 → 분석 → 결론 도출의 흐름 재현) → 결과와 의미 1문장
- 어울리는 활동: 과정 자체가 평가 대상인 활동(프로젝트, 탐구, 단계형 수행)
- 유의점: 과정 나열이 되지 않도록 전환점(어려움과 해결)을 하나 짚는다
- 사용 실적: 없음

## P-03 역량투영형
- 구성: 활동 개요 → 핵심 수행 1~2문장 → 그 수행이 드러내는 역량 서술(역량 명칭 직접 언급 금지, 행동으로 표현) → 확장 가능성 1문장
- 어울리는 활동: 성취기준·역량 연계를 명시적으로 요구받는 경우
- 유의점: 역량 서술이 근거 없는 단정이 되지 않게 반드시 수행 문장과 짝지어 쓴다
- 사용 실적: 없음

## 패턴 추가 규칙
- 새 활동 유형에서 기존 패턴이 맞지 않으면 시연 단계에서 변형을 만들고, 채택되면 새 ID로 등재한다
- 등재 시 사용 실적 줄을 반드시 포함한다
